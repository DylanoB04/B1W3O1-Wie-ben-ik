Motivatie

Hoe blijf ik gemotiveerd?

Hoe blijf ik dan gemotiveerd? Ik zelf verlies snel mijn motivatie, gelukkig is dit tot nu toe niet van pas gekomen op deze opleiding. Ik merk wel dat ik meer gemotiveerd kan raken door deze opleiding, een paar manieren hiervan zijn:

  • Wanneer ik bezig ben met een opdracht en de code is goed en alles werkt
  • Wanneer er een probleem in de code bevind
  • Wanneer ik een bepaalde oplossing vind voor een probleem
  • Als mijn opdracht optijd af is
  • Wanneer de feedback van de opdracht goed is
  • Denken aan mijn toekomstig bedrijf
